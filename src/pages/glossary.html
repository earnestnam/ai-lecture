AI 용어 사전 📚

⏱ 참고용

강의에서 만나는 AI 용어들을 비개발자 언어로 쉽게 설명합니다. 모르는 용어가 나오면 검색해보세요.

🧠 AI 기초
인공지능 AI, Artificial Intelligence
사람처럼 생각하고 판단하는 능력을 컴퓨터가 갖도록 만든 기술의 총칭이에요.
💡 비유: 사람이 아니지만 사람처럼 대화하고, 글을 쓰고, 이미지를 만들 수 있는 '디지털 두뇌'
LLM Large Language Model, 대규모 언어 모델
방대한 양의 텍스트를 학습해서 사람처럼 글을 이해하고 생성할 수 있게 된 AI 모델이에요. ChatGPT, Claude, Gemini가 모두 LLM 기반이에요.
💡 비유: 수억 권의 책을 읽고 언어를 마스터한 AI
프롬프트 Prompt
AI에게 보내는 지시·질문·요청 텍스트예요. 어떻게 부탁하느냐에 따라 AI의 답변 품질이 크게 달라져요.
💡 비유: AI에게 보내는 업무 지시서
할루시네이션 Hallucination, 환각
AI가 사실이 아닌 내용을 마치 사실인 것처럼 자신있게 말하는 현상이에요. AI의 가장 큰 주의사항 중 하나예요.
💡 비유: 자신있게 틀린 말을 하는 것. 중요한 정보는 반드시 원출처 확인 필수!
컨텍스트 윈도우 Context Window
AI가 한 번의 대화에서 기억할 수 있는 내용의 최대 길이예요. 이 범위를 넘으면 이전 내용을 잊어버려요.
💡 비유: AI의 단기 기억. 너무 긴 내용은 기억이 끊김
토큰 Token
AI가 텍스트를 처리하는 단위예요. 영어는 단어 하나, 한국어는 약 1~3글자가 하나의 토큰에 해당해요. AI 사용 요금이나 처리 한도를 토큰 단위로 측정해요.
💡 비유: AI가 읽고 쓰는 글자 묶음 단위
🤖 AI 모델
Claude 클로드
Anthropic이 만든 AI 어시스턴트예요. 긴 문서 처리와 세밀한 글쓰기에 특히 강하며, 안전하고 신중한 응답으로 유명해요.
ChatGPT 챗지피티
OpenAI가 만든 가장 널리 알려진 AI 챗봇이에요. 2022년 말 출시 이후 AI 붐을 이끌었고, 이미지 생성(DALL-E)과 다양한 플러그인을 지원해요.
Gemini 제미나이
Google이 만든 AI예요. Google 검색, Gmail, Docs 등 Google 서비스와 깊이 통합되어 있고 실시간 웹 검색이 가능해요.
모델 버전 GPT-4, Claude 3.5, Gemini 1.5 등
AI의 성능 버전이에요. 숫자가 높을수록 더 최신, 더 성능이 좋은 버전이에요. 같은 회사의 AI도 버전에 따라 능력이 다를 수 있어요.
🛠️ 도구·기능
시스템 프롬프트 System Prompt
AI에게 미리 설정해두는 업무 지침이에요. Claude 프로젝트의 "Instructions" 또는 ChatGPT의 "Custom Instructions"가 여기에 해당해요.
💡 비유: 직원에게 미리 건네주는 업무 매뉴얼
NotebookLM
Google이 만든 문서 기반 AI 연구 도우미예요. 내가 올린 문서만을 기반으로 질문에 답하고 요약을 생성해요. 팟캐스트(오디오) 형식으로도 내보낼 수 있어요.
Apps Script
Google Sheets, Gmail, Calendar 등 Google 서비스를 자동화하는 도구예요. AI에게 코드를 작성해달라고 하면 직접 코딩하지 않아도 사용할 수 있어요.
바이브 코딩 Vibe Coding
코드를 직접 짜는 대신, AI에게 원하는 것을 말로 설명해서 소프트웨어를 만드는 새로운 개발 방식이에요.
💡 비유: 요리 레시피를 직접 쓰는 대신, 셰프(AI)에게 "이런 요리 만들어줘"라고 말하는 것
멀티모달 Multimodal
텍스트뿐 아니라 이미지, 음성, 동영상 등 여러 형태의 입력을 처리할 수 있는 AI 기능이에요. Gemini가 이 분야에서 강점을 가지고 있어요.
💡 비유: 글뿐 아니라 사진, 소리도 이해하는 AI
플러그인 / 확장 프로그램 Plugin / Extension
기존 프로그램에 추가 기능을 더해주는 소프트웨어 모듈이에요. 크롬에 Claude 확장 프로그램을 설치하면 브라우저에서 바로 AI를 쓸 수 있어요.
⚙️ 기술 용어
API Application Programming Interface
서로 다른 프로그램이 서로 통신하는 방법이에요. AI API는 우리가 만든 서비스에 ChatGPT나 Claude의 기능을 넣을 때 사용해요.
💡 비유: 음식점 주방(AI)과 홀(우리 서비스) 사이의 주문 전달 시스템
파인튜닝 Fine-tuning
이미 학습된 AI 모델에 특정 분야의 추가 데이터를 학습시켜 더 전문적으로 만드는 기술이에요.
💡 비유: 일반 의사를 심장 전문의로 추가 교육시키는 것
로컬스토리지 localStorage
웹 브라우저가 사용자 컴퓨터에 데이터를 저장하는 방식이에요. 이 강의 사이트에서 체크리스트 완료 상태를 저장할 때 사용해요.
💡 비유: 웹사이트가 내 컴퓨터에 남기는 메모
생성 AI Generative AI
텍스트, 이미지, 음악 등 새로운 콘텐츠를 만들어낼 수 있는 AI예요. ChatGPT, Claude, Midjourney 등이 모두 생성 AI에 속해요.
RAG Retrieval-Augmented Generation
AI가 답변할 때 외부 문서(데이터베이스, 파일 등)를 검색해서 정보를 보완하는 기술이에요. NotebookLM이 이 방식으로 작동해요.
💡 비유: 시험 볼 때 교과서를 참고할 수 있는 '오픈북 시험'처럼 AI가 문서를 참조
