# 스킬: content-writer

## 역할
골격만 있는 빈 HTML 페이지에 비개발자 톤의 본문 콘텐츠를 채운다.

## 트리거
- 페이지 골격 생성 완료 직후 자동
- "콘텐츠 작성해줘", "내용 채워줘"

## 입력
- 골격 HTML 파일 (빈 `<article class="content-body">` 포함)
- `references/tone-guide.md` — 글쓰기 가이드
- 페이지 주제 및 핵심 메시지

## 출력
- 콘텐츠가 채워진 HTML 파일

## 콘텐츠 품질 기준
1. 비개발자가 이해할 수 있는 톤 (tone-guide.md 준수)
2. 구체적 업무 예시 3개 이상
3. 실습 과제 또는 체크리스트 포함
4. 팁 박스 1개 이상
5. 스크린샷 자리표시자 `[📸 스크린샷: 설명]` 적절히 삽입

## 자기 검증 체크
작성 후 스스로 확인:
- [ ] 기술 용어를 쉽게 풀어 설명했는가?
- [ ] 예시가 실제 업무 상황과 연관되는가?
- [ ] 독자가 읽고 바로 따라할 수 있는가?
- [ ] 도입부 → 핵심 내용 → 실습 순서인가?
